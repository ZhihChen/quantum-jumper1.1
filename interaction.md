# 量子跃迁者 - 交互设计文档

## 核心交互机制

### 维度切换系统
- **数字键1-4**：快速切换到对应维度
- **鼠标滚轮**：循环切换维度
- **空格键**：在当前维度和前一个维度间快速切换
- **Shift键**：临时进入量子叠加状态（有限时间）

### 角色控制
- **WASD/方向键**：基础移动控制
- **鼠标移动**：精确调整角色位置
- **左键点击**：与特殊物体交互
- **右键点击**：激活维度特殊能力

### 游戏界面交互
- **暂停菜单**：ESC键打开，包含继续游戏、重新开始、设置选项
- **维度指示器**：屏幕边缘显示当前维度状态
- **能量条**：显示量子能量，用于特殊能力
- **关卡进度**：显示当前关卡和收集品数量

## 多轮交互循环

### 游戏主循环
1. **观察环境** → 识别当前维度和障碍物
2. **制定策略** → 决定使用哪个维度的能力
3. **执行操作** → 切换维度并移动角色
4. **反馈调整** → 根据结果调整策略

### 挑战解决循环
1. **发现问题** → 识别无法通过的障碍
2. **思考方案** → 考虑不同维度的组合使用
3. **尝试解决** → 执行维度切换和操作
4. **学习经验** → 成功后总结规律

### 进阶学习循环
1. **掌握基础** → 熟练单个维度的操作
2. **组合运用** → 学会维度间的配合
3. **创新应用** → 发现新的解决方案
4. **精通技巧** → 掌握高级技巧和捷径

## 交互反馈系统

### 视觉反馈
- **维度切换**：屏幕边缘闪烁对应维度颜色
- **成功操作**：角色发出光芒，粒子效果
- **危险警告**：屏幕震动，红色边框提示
- **收集物品**：发光的轨迹飞向能量条

### 音效反馈
- **维度切换**：独特的量子音效
- **移动音效**：空间扭曲的声音
- **成功音效**：清脆的量子共振声
- **失败音效**：沉闷的能量消散声

### 触觉反馈（可选）
- **支持手柄震动**：在维度切换和碰撞时提供反馈

## 学习曲线设计

### 新手引导
- **第1关**：基础移动和单个维度介绍
- **第2关**：简单的维度切换练习
- **第3关**：收集品和危险区域介绍

### 渐进式挑战
- **早期关卡**：明确的解决方案，建立信心
- **中期关卡**：多种解决方案，鼓励探索
- **后期关卡**：开放式挑战，考验创造力

### 帮助系统
- **提示系统**：卡关时提供 subtle 的提示
- **演示模式**：可以观看关卡的示范通关
- **练习区域**：安全的维度切换练习环境
